# TCP Port Listener
## 🛡️ Overview
This project establishes a local server that listens for incoming TCP connections. It simulates the foundational behavior of Command and Control (C2) infrastructure used in security research.

## 🚀 Features
* Socket programming in Python.
* Handles incoming data streams.
* Demonstrates how ports remain open for communication.-C2-Basics-
